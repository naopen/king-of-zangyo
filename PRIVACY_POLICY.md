最終更新日: 2025年11月19日

## 日本語

### 概要
KING OF ZANGYO（以下「本拡張機能」）は、King of Timeの勤怠管理画面に累計残業時間を自動表示するChrome拡張機能です。本拡張機能は、ユーザーのプライバシーを尊重し、最小限のデータのみを扱います。

### 収集する情報
本拡張機能は以下の情報のみを扱います：

1. **設定情報**
   - 残業時間表示の有効/無効設定
   - この設定はChromeの同期ストレージ（chrome.storage.sync）に保存されます

2. **勤怠データの読み取り**
   - King of Timeのページ上に表示されている勤怠情報を読み取り、残業時間を計算します
   - **重要**: この情報は計算のみに使用され、保存されません

### データの使用目的
収集した情報は以下の目的でのみ使用されます：

- 残業時間表示機能の有効/無効設定の保存と復元
- King of Timeのページ上での残業時間の計算と表示

### データの保存と共有
- **保存場所**: すべてのデータはユーザーのブラウザ内のローカルストレージにのみ保存されます
- **第三者への共有**: 本拡張機能は、いかなるデータも第三者と共有しません
- **外部送信**: 本拡張機能は、いかなるデータも外部サーバーに送信しません

### 使用する権限
本拡張機能は以下のChrome API権限を使用します：

- **storage**: ユーザーの設定（表示の有効/無効）を保存するため
- **activeTab**: King of Timeのページにアクセスして残業時間を計算・表示するため

### セキュリティ
本拡張機能は、以下のセキュリティ対策を実施しています：

- データはすべてローカルで処理され、外部に送信されません
- 最小限の権限のみを要求します
- King of Timeの公式サイト（https://s2.ta.kingoftime.jp/admin/*）でのみ動作します

## English

### Overview
KING OF ZANGYO ("this extension") is a Chrome extension that automatically displays cumulative overtime hours on the King of Time attendance management screen. This extension respects user privacy and handles only minimal data.

### Information We Collect
This extension handles only the following information:

1. **Settings Data**
   - Enable/disable setting for overtime display
   - This setting is stored in Chrome's sync storage (chrome.storage.sync)

2. **Attendance Data Reading**
   - Reads attendance information displayed on the King of Time page to calculate overtime hours
   - **Important**: This information is used only for calculation and is not stored

### Purpose of Data Use
Collected information is used only for the following purposes:

- Saving and restoring the enable/disable setting for overtime display
- Calculating and displaying overtime hours on the King of Time page

### Data Storage and Sharing
- **Storage Location**: All data is stored only in local storage within the user's browser
- **Third-party Sharing**: This extension does not share any data with third parties
- **External Transmission**: This extension does not transmit any data to external servers

### Permissions Used
This extension uses the following Chrome API permissions:

- **storage**: To save user settings (enable/disable display)
- **activeTab**: To access the King of Time page and calculate/display overtime hours

### Security
This extension implements the following security measures:

- All data is processed locally and not transmitted externally
- Requests only minimal permissions
- Operates only on the official King of Time site (https://s2.ta.kingoftime.jp/admin/*)
